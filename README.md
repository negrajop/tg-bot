# Telegram Mood Tracker

Система мониторинга самочувствия пользователя с использованием Telegram-бота, базы данных SQLite, API-сервера и модуля визуализации.

## Описание проекта

Проект позволяет:

- собирать данные о самочувствии через Telegram-бота
- сохранять ответы в SQLite
- получать данные через API
- строить график изменения состояния пользователя

## Архитектура

Пользователь → Telegram-бот → БД → API-сервер → Клиент → График

## Технологии

- Python
- aiogram
- SQLite
- FastAPI
- requests
- matplotlib

---

## ▶️ Запуск проекта (несколько терминалов)

Для корректной работы системы необходимо запустить сервер и Telegram-бота в отдельных терминалах.

---

### 🔹 Шаг 1. Установка переменных окружения

Перед запуском убедитесь, что задан `BOT_TOKEN`:

#### Linux / macOS:

```bash
export BOT_TOKEN="your_telegram_bot_token"
```

#### Windows PowerShell:

```powershell
setx BOT_TOKEN "your_telegram_bot_token"
```

---

### 🔹 Шаг 2. Запуск API-сервера (Терминал №1)

```bash
uvicorn server:app --reload
```

После запуска сервер будет доступен по адресу:

```text
http://127.0.0.1:8000
```

---

### 🔹 Шаг 3. Запуск Telegram-бота (Терминал №2)

```bash
python bot.py
```

Бот начнёт принимать команды и сохранять данные в базу.

---

### 🔹 Шаг 4. (Опционально) Построение графика (Терминал №3)

```bash
python client.py
```

После выполнения:

* откроется окно с графиком
* создастся файл `mood_chart.png`

---

## 🔄 Последовательность работы

1. Запустить сервер
2. Запустить Telegram-бота
3. Взаимодействовать с ботом (в Telegram)

---

## ⚠️ Примечания

* Сервер должен быть запущен **до клиента**
* Бот и сервер могут работать параллельно
* Все компоненты работают локально (`localhost`)
